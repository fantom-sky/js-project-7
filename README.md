# JS-Project-7

Хатинка лапок — це вебзастосунок, створений для демонстрації роботи притулку для тварин.
Основна мета проєкту — допомогти користувачам знайти домашнього улюбленця та спростити процес подачі заявки на знайомство з твариною.

### На сайті можна
- переглянути список тварин, доступних для адопції
- прочитати відгуки людей, які вже знайшли собі улюбленців
- дізнатися більше про кожну тварину
- залишити заявку на знайомство через форму замовлення
- проєкт також демонструє роботу з API, динамічне створення контенту та інтерактивні UI-компоненти.

## Технології присутні у проєкті

- HTML5 - розмітка  
- CSS3 - стилізація
- JavaScript (ES6+)
- Vite — збірка та запуск проєкту
- Swiper.js — реалізація слайдера
- Axios — HTTP-запити до API
- BasicLightbox — модальні повідомлення
- REST API — отримання даних про тварин, відгуки та створення замовлень
 

## Деплой

- Клонуйте репозиторій:
```bash 
git clone https://github.com/IlliaKumpan/js-project-7.git
```
- Перейдіть у папку проєкту:

```Bash
cd js-project-7
```
- Встановіть залежності:

```Bash
npm install
```
- Запустіть режим розробки:
```Bash
npm start
```
Після цього проєкт буде доступний за адресою http://localhost:1234 (або іншою, вказаною в терміналі).

Збірка для продакшну:

```Bash
npm run build
```
